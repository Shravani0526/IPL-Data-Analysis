# IPL-Data-Analysis

##  Overview
This project analyzes IPL cricket match data using Python.

##  Objectives
- Find most winning teams
- Analyze toss decisions
- Compare match results

##  Technologies Used
- Python
- Pandas
- Matplotlib

##  Features
- Bar chart of team wins
- Pie chart of toss decisions
- Venue analysis

##  Dataset
Custom sample IPL dataset included
